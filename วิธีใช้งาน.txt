1 ดาวน์โหลด python install ให้เรียบร้อย 
2. คลิ๊กที่ install รอติดตั้งเเพ็คเกจจนเสร็จ
3. คลิ๊กที่ start  รอโปรเเกรมขึ้นให้ใส่ token ตัวเอง 
   จะขึ้นว่า token : เเละให้เราใส่Token จากนั้น กด enter
4. จะขึ้น status : ให้ใส่สเตตัสที่อยากให้เเสดง กด enter 

ถ้าโปรเเกรมทำงานสำเร็จจะขึ้น รูปสเตตัสสตรีมในหน้าโปรเเกรม


คำเเนะนำ 
อย่าปิดโปรเเกรม เพราะถ้าปิดสเตตัสก็จะหาย
เเนะนำให้ปิดออน 
ถ้าอยากให้โชว์เเถบขวา กำลังถ่ายทอดสดบน........    ให้ลบสเตตัสปกติที่ตั้งในดิสคอร์ดออก เเต่สามารถใส่อีโมจิตรงสเตตัสปกติได้
